      构筑工具

      在我最近的一次外出探险中，我似乎意外地找到了使用媒质操纵物质本身的方法。具体而言，是物质的创造，或者说是将空气粒子在身旁重整所产生的幻觉。然而，实际的操作难于登天，因此我设计了一些简单步骤用以简化。


      只需将意识集中在单个概念，比如工具之类的，媒质貌似就会处理好剩下的一切。我为它们起了个恰当的名字——“许愿”。

      每个愿望的媒质消耗量似乎没有区别，即每点耐久度消耗大约 1/10 个紫水晶粉，再额外消耗 2 个。


      矿工之愿 (player, number →)

      • Renders
        • Palettes
          浏览器不支持图案的可视化。图案代码见此：wwdwaqwqwqawq

          接受一个玩家实体和一个数。并构筑一柄镐给予目标，其耐久度即为所给数。
